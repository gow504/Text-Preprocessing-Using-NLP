# Text Preprocessing Using NLP

## Overview

This project demonstrates essential **Natural Language Processing (NLP)** text preprocessing techniques used to clean and transform raw text into a structured format suitable for machine learning and deep learning models. It serves as a foundational step for NLP tasks such as sentiment analysis, text classification, spam detection, and chatbot development.

---

## Features

* Text Cleaning
* Lowercase Conversion
* Removal of Punctuation
* Removal of Special Characters
* Tokenization
* Stop Word Removal
* Stemming
* Lemmatization
* Text Normalization
* Preparation of text for Machine Learning models

## Project Workflow

1. Load Raw Text Data
2. Clean Text
3. Convert Text to Lowercase
4. Remove Punctuation and Special Characters
5. Tokenize Sentences and Words
6. Remove Stop Words
7. Apply Stemming
8. Apply Lemmatization
9. Generate Cleaned Text for NLP Models
